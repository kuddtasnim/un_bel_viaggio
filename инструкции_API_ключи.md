# Как получить API ключи — пошаговые инструкции

---

## Часть 1 — Hugging Face токен (полностью бесплатно, карта не нужна)

Hugging Face — это платформа с тысячами бесплатных AI моделей. Токен нужен, чтобы наш сервер мог отправлять запросы к AI для блока Plan.

### Шаги:

**1. Зайди на сайт**  
Открой https://huggingface.co и нажми **Sign Up** (правый верхний угол).

**2. Зарегистрируйся**  
Введи email и пароль. Подтверди email — придёт письмо.

**3. Перейди в настройки**  
После входа нажми на аватар (правый верхний угол) → **Settings**.

**4. Открой раздел токенов**  
В левом меню найди **Access Tokens**.

**5. Создай токен**  
Нажми кнопку **New token**:
- Name: `un_bel_viaggio` (любое название)
- Role: выбери **Read** (этого достаточно для API запросов)
- Нажми **Generate a token**

**6. Скопируй токен**  
Токен выглядит так: `hf_xxxxxxxxxxxxxxxxxxxxxxxx`  
⚠️ Скопируй его сразу и сохрани — он показывается только один раз. Если потеряешь, просто создашь новый.

### Что даёт бесплатный тариф:
- До 30 000 запросов в месяц к большинству моделей
- Для личного сайта этого более чем достаточно
- Карта не нужна, подписка не нужна

## Часть 2 — Google Maps API ключ

Это немного сложнее, но я разберу каждый шаг. Google даёт **$200 кредитов в месяц бесплатно** — для личного сайта этого хватает с огромным запасом (обычно расход составляет $1–5 в месяц).

⚠️ **Карта нужна** — Google просит её для верификации аккаунта, но деньги не списываются, пока ты не превысишь $200. Для нашего сайта этого не произойдёт.

---

### Шаг 1 — Создай Google аккаунт (если нет)
Нужен обычный Gmail аккаунт. Если он уже есть — переходи к шагу 2.

---

### Шаг 2 — Войди в Google Cloud Console
Открой https://console.cloud.google.com  
Войди через свой Google аккаунт.

---

### Шаг 3 — Прими условия и настрой биллинг

При первом входе Google попросит принять условия использования и привязать карту.

Нажми **Activate** или **Попробовать бесплатно**:
- Выбери страну
- Введи данные карты (только для верификации — $0 не списывается сразу)
- Нажми **Начать бесплатный период**

Ты получишь $300 пробных кредитов на 90 дней + $200 постоянных кредитов в месяц после.

---

### Шаг 4 — Создай проект

В верхней панели нажми на выпадающее меню рядом с логотипом Google Cloud → **Новый проект**:
- Название: `un-bel-viaggio`
- Нажми **Создать**

Подожди несколько секунд — проект создаётся.

---

### Шаг 5 — Включи нужные API

В левом меню выбери **API и сервисы** → **Библиотека**.

Найди и включи (нажать «Включить» для каждого):

1. **Maps JavaScript API** — основная карта
2. **Places API** — поиск мест по названию и адресу
3. **Geocoding API** — перевод адреса в координаты (нужен для маркеров)

---

### Шаг 6 — Создай API ключ

В левом меню: **API и сервисы** → **Учётные данные** → кнопка **Создать учётные данные** → **Ключ API**.

Ключ создастся мгновенно. Он выглядит так: `AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

---

### Шаг 7 — Ограничь ключ (важно для безопасности)

Нажми на название только что созданного ключа → откроется страница настройки.

**Ограничения по приложениям:**
- Выбери **HTTP-реферреры (веб-сайты)**
- Добавь адрес твоего будущего сайта: `*.up.railway.app/*`
- И для локальной разработки: `localhost/*`

**Ограничения по API:**
- Выбери **Ограничить ключ**
- Выбери только три API, которые мы включили: Maps JavaScript API, Places API, Geocoding API

Нажми **Сохранить**.

---

### Шаг 8 — Скопируй ключ

Вернись на страницу «Учётные данные» и скопируй ключ. Он выглядит так:  
`AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

## Что делать с ключами

Когда получишь оба — напиши их мне в чат (или сохрани в надёжном месте). Я встрою их в конфигурацию сервера в виде переменных окружения — они не попадут в GitHub и не будут видны в коде.

Никогда не публикуй эти ключи открыто — не вставляй в GitHub, не пиши в комментариях к коду.
